git命令

git init  初始化

git status  检测当前目录下文件的状态，文件是否已经被git管理起来

三种状态的变化
红色 ： 代表未被管理的文件或修改过的文件; git add '文件名' ； git add . （.管理全部文件）
绿色:git已经管理起来 
（第一次，需要配置用户信息，否则会报错。
Run

  git config --global user.email "you@example.com"
  git config --global user.name "Your Name"

git commit -m ‘描述信息’
生成版本

git log 查看版本信息
